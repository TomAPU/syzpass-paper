\section*{Open Science}
Our code and data are available at the following link \\ \url{https://osf.io/vgreh/}

Inside the artifact archive, the \texttt{Syzpass} folder contains the source code and instructions required to replicate our experiments. The \texttt{cases} folder includes all benchmark cases used to evaluate SyzPass, and the \texttt{data} folder provides the result data generated by SyzPass for each case. For ethical reasons, we do not release exploit code.
